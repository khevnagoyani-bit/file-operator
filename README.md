# file-operator

📖 Personal Journal Manager

«Capture your thoughts, preserve your memories, and organize your daily reflections—all from your terminal.»

Personal Journal Manager is a Python-based command-line application that provides a simple and efficient way to maintain a digital journal. Users can create journal entries with automatic timestamps, review previous notes, search for specific memories, and manage their journal through an intuitive menu-driven interface.

---

🌟 Overview

Writing a journal is one of the best ways to record experiences, ideas, goals, and personal growth. This application brings that experience to the command line by allowing users to securely store text entries in a local file while keeping the interface clean and beginner-friendly.

The project demonstrates practical file handling techniques in Python while introducing users to real-world programming concepts.

---

✨ Key Features

- 📝 Create unlimited journal entries
- 🕒 Automatically records the current date and time for every entry
- 📚 View all saved journal entries
- 🔍 Search journal entries using keywords or dates
- 🗑️ Delete all entries with confirmation
- 📂 Automatically creates and manages a local journal file
- 💻 Interactive menu-driven interface
- ⚡ Lightweight and easy to use
